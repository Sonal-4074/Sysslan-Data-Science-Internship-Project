# Indian Railway Data Analysis and Journey Time Prediction System

## Overview

This project was completed as part of my Data Science Internship at Sysslan IT Solutions.

The objective of this project is to analyze Indian Railway data and develop a machine learning model capable of predicting journey duration based on train route characteristics.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* Jupyter Notebook

## Project Structure

### Level 1 – Data Understanding

* Dataset Loading
* Data Exploration
* Statistical Summary
* Missing Value Analysis

### Level 2 – Data Preprocessing

* Data Cleaning
* Time Formatting
* Journey Duration Calculation
* Feature Preparation

### Level 3 – Exploratory Data Analysis

* Longest Route Analysis
* Station Traffic Analysis
* Route Pattern Analysis
* Insight Generation

### Level 4 – Data Visualization

* Journey Duration Distribution
* Train Traffic Visualization
* Distance vs Journey Duration Analysis
* Visualization Interpretation

### Level 5 – Machine Learning

* Feature Engineering
* Train-Test Split
* Linear Regression Model Training
* Model Evaluation

### Level 6 – Prediction System

* Journey Time Prediction System
* Actual vs Predicted Analysis
* Result Visualization

## Machine Learning Features

The Linear Regression model uses:

* Total Distance
* Number of Stops

to predict:

* Journey Duration (Hours)

## Key Outcomes

* Performed complete data analysis workflow
* Created meaningful visualizations
* Built a journey time prediction model
* Compared actual and predicted journey durations
* Developed an end-to-end predictive analytics project
